throwdis コマンドを提供します。 物を投げたときの

投げた初速度
投げた角度
投げた高さ
地面の傾斜
から，重力加速度0.906 m/s^2 においての

到達する水平距離
落下する時間
落下したときの進行角度 を出力します.
リポジトリの入手
リポジトリを保存したいディレクトリにて，

$ git clone https://github.com/peanutnuts/Throwing-Distance-Calculator  
と入力してください。

使い方
$ cd Throwing-Distance-Calculator
$ ./throwdis <投げた初速度(m/s)> <投げた角度(deg)> <投げた高さ(h)> <地面の傾斜(deg)>
とそれぞれ半角英数字で値を代入して入力してください。 入力された値から計算を行い，以下を出力し，コマンドを終了します。

<到達距離(m)>
<落下時間(s)>
<落下時の進行角度(deg)>
※　もし，60s 以上になっても，仮想上の投擲物の座標が地面より下にならなかった場合，以下の文章が出力されコマンドを終了します。

飛翔時間が60秒を超過。計算を終了します。
※　投げる角度，地面の傾き，の二つについては，水平方向を0，そこから見上げる時は正の数，見下げる時は負の数で角度を入力してください。 ※　地面の傾きについては 85(deg) ~ -85(deg) の範囲で入力してください。 ※　重力加速度は9.806 m/s^2 で固定です。 ※　結果にはわずかながら誤差が生じます。 ※　地面に到達するときの速度が早すぎると，結果の誤差が大きくなります。

動作確認環境
Ubuntu 22.04.5 LTS Python 3.13.5 git version 2.34.1
