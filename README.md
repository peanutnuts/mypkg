#説明
deg-rad.py ノードを提供します。 

角度deg を角度rad に変換して出力します。

入力
- /angle_deg   (std_msgs/Float64)

出力
- /angle_rad   (std_msgs/Float64)

動作
- rad = deg * π / 180

#入手方法
- $ git clone git@github.com/peanutnuts/mypkg.git  
と入力してください。

#動作確認環境
- Ubuntu 22.04.5 LTS 
- Python 3.13.5  
- git version 2.34.1
